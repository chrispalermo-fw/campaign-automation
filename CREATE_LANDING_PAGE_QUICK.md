# Create Landing Page - Quick Guide

Since your form is ready, creating the landing page manually is the fastest way!

## 3-Minute Setup

### Step 1: Create Landing Page

1. **Go to HubSpot** ‚Üí **Marketing** ‚Üí **Landing Pages** ‚Üí **Create landing page**

2. **Choose a template**:
   - Select any template (or start from scratch)
   - You'll customize it anyway

3. **Page Settings**:
   - **Page name**: `Campaign Automation Tool`
   - **URL slug**: `campaign-automation` (or leave default)
   - **Page title**: "Campaign Automation Tool - Create Campaigns"

### Step 2: Add Content

1. **Add a heading**:
   - Drag "Rich Text" or "Heading" module
   - Add: `üöÄ Campaign Automation Tool`

2. **Add description**:
   - Add another text module with:
     ```

     Fill out the form below to automatically create:
     ‚Ä¢ Campaign in HubSpot
     ‚Ä¢ Campaign in Salesforce
     ```

### Step 3: Add Your Form

1. **Drag "Form" module** onto the page
2. **Select form**: Choose **"Campaign Automation Form"**
3. The form will appear on the page

### Step 4: Customize & Publish

1. **Style the page** to match your brand (colors, fonts, etc.)
2. **Preview** to make sure it looks good
3. **Publish** the page
